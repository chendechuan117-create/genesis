# Genesis v0.2.0 - 已知问题

## 🐛 当前问题

### 1. 偶尔达到最大迭代次数

**现象**:
- 用户问"你记得我吗"
- AI 反复调用工具（shell, list_directory）
- 达到 15 次迭代限制后报错

**原因**:
- AI 有时会把简单问题理解成"需要收集系统信息"
- DeepSeek 模型的行为有随机性
- 上下文和记忆内容会影响 AI 的判断

**临时解决方案**:
1. 增加了 max_iterations (10 → 15)
2. 简化了 system prompt
3. 改进了错误提示

**根本解决方案**（需要时间）:
- 通过大量真实对话训练
- 持续优化 system prompt
- 可能需要 few-shot examples

**用户建议**:
- 如果遇到这个错误，换个方式问
- 例如："你记得我吗" → "我是谁"
- 或者重新开始对话

---

### 2. 回复有时还是太啰嗦

**现象**:
- AI 会说"我回来了"、"根据记忆"
- 有时会列举能力清单

**原因**:
- System prompt 虽然简化了，但 AI 还是有自己的习惯
- 需要通过自适应学习逐渐改进

**解决方案**:
- 继续使用，AdaptiveLearner 会学习你的偏好
- 如果不满意，可以直接说"太啰嗦了"，它会学习

---

### 3. 工具描述还不够清晰

**现象**:
- AI 有时不知道什么时候该用工具

**计划**:

- 学习 OpenClaw 的工具设计
