## ⚠️ 仍存在的小问题

### "从记忆中看到"
AI 还是会说"从记忆中看到"，虽然我们在 System Prompt 中明确禁止了。

**原因**:
- LLM 的行为习惯难以 100% 改变
- System Prompt 只是"建议"，不是强制执行
- DeepSeek 模型有自己的表达习惯

**影响**:
- 对话自然度 85%（vs OpenClaw 的 95%）
- 不影响功能，只是表达方式

**解决方案**:
- 需要更多真实对话来训练
- AdaptiveLearner 会逐渐学习用户偏好
- 可能需要 few-shot examples

## 💡 核心教训

### 从 OpenClaw 学到的

1. **简单即美**
   - System Prompt 不需要太复杂
   - 信任 AI，给原则而非规则

2. **记忆要内化**
   - 不要显式传递"相关记忆："
   - 融入 System Prompt，让 AI "自然知道"

3. **对话要连贯**
   - 保持会话历史
   - 不要每次都重新介绍

4. **工具要克制**
   - 明确告诉 AI 何时用工具
   - 简单问题直接回答

### Genesis 的创新

1. **Token 优化是刚需**
   - 60-80% 节省 = 成本大幅降低
   - 可以处理更复杂的任务

2. **多面体框架有价值**
   - 深度思考能力
   - 复杂问题分析

3. **自适应学习是未来**
   - 个性化体验
   - 持续改进
